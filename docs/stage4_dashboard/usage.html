使用说明

如何阅读这个进度网站

这个网站不是论文最终结果页，而是 Plan B 实验、论文数字和 source-of-truth 的协作看板。

最重要原则 看状态，再看数据，再看风险 不要直接把未标为可写的数字放进论文。

一眼看懂

推荐阅读顺序
1. 当前状态

看首页顶部卡片和图表，了解数据池、候选数、标注数和当前主要实验进展。

2. 论文写作数据

这里告诉你某一节论文目前能写什么数字，点“来源”会打开可视化数据表。

3. 方案完整数据清单

这里逐项对照方案要求的表。重点看“数据状态”和“论文表状态”。

4. 当前风险

这里列出不能直接写进论文的坑，例如真实 audit agreement、raw CC3M 重复率等。

状态含义

合作者写作时先看这里
前端显示 含义 论文写作建议
已完成 相关 source-of-truth 数据已经存在。 仍需看“论文表状态”和风险提示，不等于表已经写完。
部分完成 已有一部分数字，但缺关键字段或还只是估算。 只能写设计、趋势或带限定的初步结果。
待开始 核心实验或数据还没有。 不能写结果，只能写实验设计。
数据可写 / 表未成稿 数据已经在前端可见，但还没有整理成论文表。 可以据此起草表格，但要绑定 experiment id。
缺核心数据 论文需要的关键数字还没有。 不要填占位数字，不要脑补。

高风险项
真实 inter-annotator agreement

原始 CC3M 重复率

当前 1000 条标注集来自 high-joint hard candidates，不能代表 raw CC3M 的重复比例。

如何查看数据文件

不会再直接打开 raw JSON

在“论文写作数据”“方案完整数据清单”“数据导出”中点击来源，会进入一个数据查看页。 CSV 会显示为表格；JSON 会显示为结构化卡片。页面顶部会保留原始文件路径，方便追溯。
